C.a.R.

You have downloaded and unzipped the documentation and all necessary files for the dynamical geometry program C.a.R. (Compass and Ruler) This is a short introduction into the package. For more information about other installation options visit the Web page of C.a.R.

Starting the Program

This installer does not create icons for the program. You can start C.a.R. using the program archive zirkel.jar, which you find in the directory "doc" in your installation directory. On most systems, a double click on this archive should start the program. If that does not work, try the following command.

java -jar zirkel.jar

You are then running the latest release and the current examples. If you wish, use the previous stable release in the Download subdirectory of your installation directory.
