Euclidean Plane and its Relatives

The book grew from my lecture notes which I have been polishing since 2001. It is designed for a semester-long course in Foundations of Geometry and meant to be rigorous, conservative, elementary and minimalistic. We use the so called metric approach introduced by Birkhoff.

It is free to download [v8]. You can also order a printed copy at amazon or directly at createspace (use code EH8HYU54 in this case).

Please write me a short message if you are teaching a course using this book. Also, if you see a mistake, email it to me.

If you would like to contribute, you will have to use Git and LaTeX. The following command will create a local copy of the source code for you.

git clone https://github.com/anton-petrunin/birkhoff.git
